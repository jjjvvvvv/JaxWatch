Troubleshooting

Common issues and solutions:

  • File not found: Make sure JaxWatch data files exist in the expected locations
  • Permission errors: Check file permissions for reading/writing data files
  • JSON parsing errors: Verify that data files contain valid JSON
  • Document Verifier errors: Ensure Document Verifier is properly installed and configured

Expected file locations:

  • admin_ui/data/projects_index.json - Raw JaxWatch data
  • admin_ui/data/projects_enriched.json - Enhanced projects (created by Document Verifier)
  • document_verifier/demo_output.json - Demo enhancement data
  • dashboard/status.json - Dashboard status (created automatically)
